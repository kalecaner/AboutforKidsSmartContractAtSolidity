# 🧒 AboutForKidsSmartContractAtSolidity  
### AboutForKids Smart Contract in Solidity

Bu proje, **Solidity ile yazılmış basit bir akıllı sözleşme (smart contract)** örneğidir.  
Özellikle **çocuklara veya yeni başlayanlara blockchain mantığını anlatmak** için tasarlanmıştır.  
**Tamamen eğitim ve kişisel gelişim amaçlıdır.**

This project is a **simple smart contract written in Solidity**.  
It is designed mainly to **explain blockchain concepts to kids or beginners**.  
**It is purely for educational and personal development purposes.**

---

## 🎯 Projenin Amacı / Project Purpose

**TR:**  
- Solidity ile akıllı sözleşme geliştirmeye giriş yapmak  
- Basit fonksiyonlar, veri depolama ve olay (event) kullanımlarını göstermek  
- Eğitim ortamında kullanılabilecek bir örnek oluşturmak  
- Gerçek dünyadaki uygulamalar için güvenlik içermeyen basit bir demo sunmak  

**EN:**  
- Introduce smart contract development with Solidity  
- Show examples of simple functions, data storage, and events  
- Provide an example that can be used in an educational context  
- Serve as a demo without production-grade security  

---

## 🏛 Proje Yapısı / Project Structure

- **contracts/** → Ana akıllı sözleşme kodları  
- **scripts/** → Deploy ve etkileşim komutları  
- **tests/** → Öğrenme amaçlı testler  

---

## 🛠 Kullanılan Teknolojiler / Technologies

- Solidity (ör: ^0.8.x)  
- Hardhat / Truffle / Remix IDE (geliştirme için)  
- Ethers.js / Web3.js (etkileşim için)  
- Node.js + npm
